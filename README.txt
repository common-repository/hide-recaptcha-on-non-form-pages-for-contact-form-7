Simple little plugin that hides reCAPTCHA on pages without Forms for Contact Form 7

== Description ==

Hide reCAPTCHA on all pages except the pages with forms (like the contact page). By default Contact Form 7 shows Google reCAPTCHA on all pages, including the homepage, posts and shop pages. When we only need it on pages with Forms, like the Contact page. This plugin will look for pages with form shortcodes and show reCAPTCHA on those pages, and hide it on all other pages.

== Installation ==

1. Install the plugin from the [Wordpress Plugin Directory](https://wordpress.org/plugins/).
2. Activate it.
3. That's it. The plugin has no options, once enabled it will hide reCAPTCHA on all pages except the pages with forms.

== Dependencies ==

1. Wordpress v3.9.2 and later (Tested up to 6.0)
2. PHP v5.6.0 and later (Tested up to 7.4)
3. Contact Form 7 (Tested up to 5.5.6.1)
